# EddySims.com

This is my personal website. Its made in NextJs. The code is super ugly right now... Please don't judge me. I will be improving over time!

## Development Server

Currently, this site is build with NextJs. To start the development server run:

`npm run dev`

## Storybook

The best components are the ones that are built in isolation. To do this we use [Storybook](https://storybook.js.org/).  To start the storybook server run:

`npm run storybook`

## Linting

I currently use EsLint to list the project. To run the linter, use the command:

`npm run lint`

## Testing

I am currently using the Jest setup that is recommended by [NextJs](https://nextjs.org/docs/testing#jest-and-react-testing-library)

To run the test suite, use the command:

`npm run test`
