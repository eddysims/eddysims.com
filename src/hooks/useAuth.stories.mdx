# useAuth

The `useAuth` hook is used to check user authentication throughout the application.

## Usage

```ts
import { useAuth } from "@/hooks/useAuth";
```

## Logging a user in

To trigger a login, you will want to use the `login` function.

```tsx
import { useAuth } from "@/hooks/useAuth";

function Component() {
  const { login } = useAuth();

  return <button onClick={login}>Login</button>;
}
```

### Callbacks

The `login` function accepts 2 callbacks:

`onLogin` that will fire after the user has logged in.
`onError` that will fire if there is an error logging in.

## Logging a user out

To trigger a logout, you will want to use the `logout` function.

```tsx
import { useAuth } from "@/hooks/useAuth";

function Component() {
  const { logout } = useAuth();

  return <button onClick={logout}>Logout</button>;
}
```

### Callbacks

The `login` function accepts 1 callback1:

`onLogout` that will fire after the user has logged out.

## isLoggedIn

To simply check if the user is logged in, use the `isLoggedIn` variable returned from the hook.

```tsx
import { useAuth } from "@/hooks/useAuth";

function Component() {
  const { isLoggedIn } = useAuth();

  return <div>The user is {!isLoggedIn ? "not" : ""} logged in</div>;
}
```

## Getting a user

To get some user information, you can use the `user` object returned from the hook.

```tsx
import { useAuth } from "@/hooks/useAuth";

function Component() {
  const { user } = useAuth();

  return <div>The users name is {user?.displayName}</div>;
}
```

## Loading

The `useAuth` hook contains a `loading` state that can be used to indicate that the application is currently getting information about the session.

```tsx
import { useAuth } from "@/hooks/useAuth";

function Component() {
  const { loading, user } = useAuth();

  return (
    <div>
      {loading ? (
        <div>Loading...</div>
      ) : (
        <div>The users name is {user?.displayName}</div>
      )}
    </div>
  );
}
```
