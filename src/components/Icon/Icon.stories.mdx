# Icon

`Icon` is a wrapper around [react-feather](https://www.npmjs.com/package/react-feather) which uses [Feather Icons](https://feathericons.com/).

## Usage

```tsx
import { Icon } from "@/components/Icon";
```

## Props

<ArgsTable component={Icon} />

## Notes

- The stroke color of an `Icon` should be decided by its parent component.
